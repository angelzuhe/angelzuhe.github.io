Instalar Rsync en Windows 10

Rsync es una herramienta de linea de comandos utilizada para sincronizar archivos y directorios entre sistemas, tanto locales como remotos.

  • Puede usar rsync atravez de un protocolo SSH.
  • Realiza copias incrementales, solo transfiere aquellos ficheros que han sufrido cambios.
  • Realiza copias de seguridad de ficheros y directorios de forma recursiva.

Rsync es una herramienta que ya viene preinstalada en sistemas operativos tipo unix; Linux y MacOS, para Windows se puede usar una adaptación de rsync que se llama DeltaCopy. Aquí vamos a usar WSL (Windows Subsystem for Linux) y apartir de aquí hacer la instalación de rsync.

Activar WSL

Tan sólo hay que ir a: «panel de control» –> «Agregar o quitar programas» –> «Activar o desactivar características de Windows» y aquí activar el checkbox “Subsistema de Windows para Linux”.

El paso siguiente es instalar el Sistema Linux, desde la tienda de Microsoft hay varias disponibles. Se encuentra distribuciones GNU/Linux, como: Debian GNU/Linux, Kali Linux y Ubuntu Linux, entre otras.

Yo uso Debian por lo tanto es la que he instalado. Anque esto no es superficial ya que tanto Kali Linux como Ubuntu estan basadas en GNU/Linux Debian.

Una vez instalada WSL Debian aparece como cualquier otra aplicación disponible en el menu de inicio, o ejecuta este comando en la consola «cmd»: wsl -d Debian
